Slope Battles Instructions

Objective

The objective of the game is to place points on the grid, following the rules outlined below. Each point you place contributes to the challenge of maintaining a unique configuration on the grid.

Game Termination Conditions

  1. Placing Outside the Grid: Attempting to place a point outside the grid's confines ends the game.
  2. Point Overlap: Placing a point at an occupied location halts the game.
  3. Row Duplicates: Placing two or more points on the same row results in game termination.
  4. Column Duplicates: Similarly, placing two or more points on the same column results in game termination.
  5. Shared Slopes: Placing points that create line segments with shared slopes concludes the game.
